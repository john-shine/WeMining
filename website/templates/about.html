Technical Method

Panoramic Online Microblogging Analysis System for Network Public Opinion Monitoring: Online social media like micro-blog provides a large data set for us to play with, there are more than 250 million Chinese Internet users publish tweets through the Sina Weibo (a Twitter-alike website in China) every day and night. In the analysis,we get a large data set with the combination using of sina weibo API and Web crawler written python language.We've been trying to explore the laws of topic communicating and public opinion trend in social network and obtain user's personal interests and emotions by corpus analysis on the key nodes in the topic forwarding path.We visualize the forwarding path of a particular hot topic in the weibo with LBS(Location Based Service)tools like google map.
